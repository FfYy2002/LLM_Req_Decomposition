##  实验结果
- **分析问题**：拆分准确率，也即是大模型拆分的简单AR需求，是否与人工拆分的足够一致？
- **TP/FP/FN/TN定义**：人工和大模型拆分的 **每一条编号项** 即为 **一个简单AR需求点**
    
    | 类型 | 定义 | 对应指标 |
    |------|------|----------|
    | TP（真阳性） | 人工需求点被 LLM 正确覆盖（语义等价） | ✔️ |
    | FN（假阴性） | 人工需求点未被 LLM 覆盖（遗漏） | ❌ 漏掉 |
    | FP（假阳性） | LLM 生成了人工中没有的需求点（编造/冗余） | ❌ 多出 |
    | TN（真阴性） | 不适用 | — |
